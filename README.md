# Meridian Feedback

Meridian is a focus timer built around a live globe — helping you see where your time goes as it happens.

This repo is where you can report bugs, share ideas, and tell me what's working (or not). I read every issue personally.

## Submitting feedback

1. Click **Issues** → **New Issue**
2. Choose the bug report template or open a blank issue
3. Describe what happened — screenshots are always helpful

## A note

This is an early build. Things will break, things will change, and your feedback directly shapes what gets built next. Thank you for trying it.

— Robert, Crane Creek Studios
